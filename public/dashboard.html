Setup Instructions
1
Add Browser Source

In OBS, add a new Browser Source. Name it "ChatVibes TTS". Paste the URL copied above. You can ignore Width/Height.

2
Control Audio via OBS

Check the box that says "Control audio via OBS". Ensure "Shutdown source when not visible" is unchecked.

3
Monitor and Output

In your OBS Audio Mixer, find "ChatVibes TTS". Click the gear icon > Advanced Audio Properties. Set Audio Monitoring to "Monitor and Output" so both you and stream can hear it.
